An AI agent that solves Raven's Progressive Matrices problems

For my Knowledge Based AI course, I created a unique AI agent that solved Raven’s Progressive Matrices problems. The agent would analyze images, detect patterns within the images, and then select a corresponding image that matched some pattern. I used numpy, PIL, and various other python packages to develop a successful agent. This project taught me more about the many image analysis python applications.

Designing and improving an OS

During my Designing Operating Systems course, I completed several projects that implemented major improvements to the XV6 operating system. These improvements include copy-on-write forking additions, lazy zero-page allocation, scheduling and threading capabilities, user isolation and more. This project improved my C proficiency and taught me to think deeply about OS design choices.

Binary Black Holes and Gravitational Waves

Simulated and analyzed data from gravitational waves emitted from merging binary black holes. Headed 10+ presentations concerning gathered data and progress updates for the astrophysicsresearch department. My contributions would go on to assist further research concerning possible effects of gravitational wave emittance. This research improved my proficiency with python and exposed me to the many data analysis tools python has to offer.

MARS Mission Simulations

I supported and developed a complex simulation for potential resource harvesting for Mars missions based on mineral compositions, reactivity, and more. Collaborated with NASA JPL and GTRI scientists to support missions for acquiring renewable energy sources, dust mitigation methods, and rover prototype design. This research improved my proficiency with python and matlab.

A.I. and Autonomous Robots

I designed optimal algorithms for perception and locomotion of Georgia Tech’s A1 robot and the MIT Cheetah robot. I also developed several simulations to test predicted limb movements using pybullet and gazebo.
